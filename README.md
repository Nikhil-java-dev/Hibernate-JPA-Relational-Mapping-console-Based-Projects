# Hibernate Relationship Mapping Projects

This repository contains example projects demonstrating **Hibernate ORM relationship mappings** using Java and JPA. These projects show how different entity relationships are implemented in real-world applications.

## 📌 Projects Included

### 1. Hibernate Unidirectional Relationship Mapping

This project demonstrates **unidirectional mappings**, where only one entity knows about the relationship.

Examples included:

* One-to-One Mapping
* One-to-Many Mapping
* Many-to-Many Mapping

### 2. Hibernate Bidirectional Relationship Mapping

This project demonstrates **bidirectional mappings**, where both entities maintain references to each other.

Examples included:

* One-to-One Mapping
* One-to-Many Mapping
* Many-to-Many Mapping

## 🛠 Technologies Used

* Java
* Hibernate ORM
* JPA (Java Persistence API)
* Maven
* MySQL
* Eclipse IDE

## 📂 Project Structure

```
Hibernate-Projects
 ├── hibernate-relationship-unidirectional-mapping-crud-operation
 └── hibernate-relationship-bidirectional-mapping-crud-operation
```

## 🚀 How to Run the Project

1. Clone the repository

```
git clone https://github.com/Nikhil-java-dev/Hibernate-Projects.git
```

2. Import the project into **Eclipse IDE**.

3. Configure the database connection in `persistence.xml`.

4. Run the controller classes to test CRUD operations.
